# JWEssentials

**A modern, full-featured Essentials plugin for Endstone servers.**  
Built in Python for EndstoneMC, inspired by EssentialsX, optimized for Minecraft Bedrock Dedicated Server.

</div>

---

## ✨ Overview

JWEssentials is an all-in-one utility plugin designed for Endstone servers, providing the most important commands and systems needed for survival, economy, and server management.

It brings a complete Essentials-style experience to Bedrock servers running Endstone, including teleportation, homes, warps, private messaging, moderation utilities, player tools, and more.

---

## 🚀 Features

### 🧭 Teleportation System
Comprehensive teleport commands for players and staff:

- `/tp`
- `/tphere`
- `/top`
- `/jump`
- `/spawn`

Features:

- Safe teleport checks
- Cross-dimension support
- Permission-based teleport control
- Instant teleportation

---

### 🏠 Home System

Players can create and manage personal homes.

Commands:

- `/home`
- `/sethome`
- `/delhome`

Features:

- Multiple homes
- Configurable home limits
- VIP/unlimited permissions
- Persistent storage
- Named homes support

---

### 🌍 Warp System

Public warp management for server destinations.

Commands:

- `/warp`
- `/setwarp`
- `/delwarp`

Features:

- Global server warps
- Admin-managed warp creation
- Persistent YAML storage
- Permission control

---

### 🤝 Teleport Request System

Player-to-player teleport request system.

Commands:

- `/tpa`
- `/tpahere`
- `/tpaccept`
- `/tpdeny`
- `/tpcancel`

Features:

- Request timeout
- Safe request handling
- Anti-spam cooldown
- Incoming/outgoing request tracking

---

### 💬 Social Features

Communication utilities for players and staff.

Commands:

- `/afk`

Features:

- Private messaging
- Reply shortcut
